# Single Cycle Cpu In Logisim

## Description
A single-cycle implementation of a subset of the ARMv7 instruction set using digital logic in Logisim.

## Instructions:
In the main, the Rom Selector is what selects which program to run. 
- 00 = sum_array
- 01 = fib_iter
- 10 = fib_rec
- 11 = find_max
